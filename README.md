# clam clam
>北理工软件工程大三小学期实习作业\
一个简单的windows即时通讯软件
## 开发工具：
* Visual Studio 2019
* MySQL 8.0 数据库
* MFC 图形界面开发
## 实现功能：
* 登录
* 注册
* 群聊
* 添加好友
* 选择好友单聊

____
